Pip-Boy Customisation Framework

Pip-Boy Customisation Framework aka PBCF is a mod for Fallout 4.
It allow to modify the Pip-Boy skins in-game. It predate the Creation Club update which allowed to modify the Pip-Boy in-game.
It has been updated to support the official method but it still is usefull as it bring some more options. More Parts to modify , Misc Mod & z Dynamics Namings.
